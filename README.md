# Nest.js TODO API

## Set up the Nest.js application

### Nest.js install
```shell
$ yarn install
```

### docker
```shell
$ docker-compose up
```

### Nest.js start develop
```shell
$ yarn start:dev
```

### open Nest.js
```shell
$ open http://localhost:4000
```
